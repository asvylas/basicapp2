SEVN Stack application for listing and viewing properties: houses, apartments, others.

Front-end: Vuejs, Vuex, Vue-router, Vuetifyjs, Axios.
Back-end: Node.js, Express, Axios, Bcryptjs, Sequelize.
Database: SQLite.
